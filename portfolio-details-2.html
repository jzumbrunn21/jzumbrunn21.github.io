Service Squirrel

  • Technologies: JS, Python, React, Redux, Flask
  • Project URL: https://service-squirrel.onrender.com

A comprehensive service marketplace designed to connect users with a wide range of services such as lawn care, cleaning, and more. This platform serves as a one-stop solution for all service needs.

  • Collaborated with a team of engineers utilizing Agile methodologies and Git for version control to ensure a prompt delivery of high-quality features within a short time-span.
  • Generated a responsive front-end using React, Redux, HTML and CSS to enhance the user experience with seamless navigation and intuitive design.
  • Engineered a robust, custom database that utilizes Postgres and SQLAlchemy to effectively handle large volumes of data, ensuring high performance and reliability.
  • Designed a custom API using Flask and REST principles to provide secure and accurate navigation, ensuring data integrity and security.
  • Implemented a robust user authentication system that restricts access to user-specific data and features, leveraging custom front-end routing and back-end password hashing for secure and personalized user experiences.
